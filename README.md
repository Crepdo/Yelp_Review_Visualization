If you use vscode, use Ctrl+Shift+P,
task: run tasks > task: install dependencies > task: run task: start app

if not in vscode
run npm install in both frontend and backend.
Then run npm run start in both frontend and backend.
```ps1
cd backend
npm install
npm run start
```
```ps1
cd frontend
npm install
npm run start
```
